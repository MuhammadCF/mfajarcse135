GET /api/static Retrieve every entry logged in the static table
GET /api/static/{id} Retrieve every entry logged in the static table
POST /api/static Add a new entry to the static table
DELETE /api/static/{id}
PUT /api/static/{id} Delete a specific entry from the static table (that matches the given id)
PATCH /api/static/{id} Update a specific entry from the static table (that matches the given id)
